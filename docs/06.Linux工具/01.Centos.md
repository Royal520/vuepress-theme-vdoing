## `repotrack`是一个yum包依赖下载工具使用方法如下

```
# 首先安装yum-utils
sudo yum install yum-utils

# 使用repotrack下载nginx及其依赖项
mkdir nginx-packages
repotrack -p nginx-packages nginx

# 接下来，将nginx-packages目录复制到离线机器上
# 在离线机器上安装nginx
cd nginx-packages
sudo rpm -Uvh --force --nodeps *.rpm
```
